# 兰州市2026年事业编考试综合管理题库练习系统

一个专门为兰州市2026年事业编考试准备的在线刷题平台，包含公共基础知识、行政能力测试、综合应用能力、时事政治、法律法规等科目的练习题库。

## 功能特性

- 📚 题库练习 - 按章节分类做题
- 📝 模拟考试 - 限时模拟考试
- ❌ 错题本 - 自动收集错题
- 📊 数据分析 - 学习统计和分析
- 📱 响应式设计 - 支持手机和电脑

## 技术栈

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- React Router DOM 6
- Zustand
- Chart.js
- Lucide React

## 本地开发

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

## 部署到 Vercel

1. 访问 https://vercel.com/signup 注册账号
2. 点击 "New Project"
3. 导入此仓库
4. 自动部署成功！
